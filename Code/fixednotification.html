Demonstração Fixed Notification

Notificação fixa feita usando Angular JS e Bootstrap.


HTML:
<div data-fixed-notification=""></div>
JS:
angular.module('project', ['keeple.controls.fixed-notification', 'ngSanitize'])

['$scope', 'fixed-notification.service.helper',
    function ($scope, fixedNotificationHelper) {
$scope.sendNotification = function () {
    fixedNotificationHelper.showNotifcation($scope.message, $scope.type);
};
